# Roadmap

## M1 — Platform foundation

Repository structure, architecture, functional specification, safety policy, security baseline and knowledge schema.

## M2 — Core platform

Assistant orchestration, knowledge retrieval, safety enforcement, provider abstraction and automated tests.

## M3 — Web experience

Accessible branded chat widget for mobile and desktop.

## M4 — GetGasCert reference implementation

Approved GGC knowledge, pricing, service routing, Calendly links, WhatsApp escalation and lead capture.

## M5 — Administration and analytics

Knowledge management, conversation review, reporting and configuration controls.

## M6 — Multi-business productisation

Tenant isolation, theme configuration, deployment tooling and reusable connectors.
