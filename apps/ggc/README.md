# GetGasCert Assistant — RI-001

The first reference implementation of the EVENAI Assistant Platform.

## Intended scope

- CP42, CP44 and CP12 service guidance
- Published pricing
- Commercial gas emergency routing
- East Anglia service-area guidance
- Calendly booking handoff
- WhatsApp escalation
- Lead qualification and capture

Business-specific content and configuration must remain isolated from reusable platform packages.
