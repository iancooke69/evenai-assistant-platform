# Functional Specification

## Product

EVENAI Assistant Platform.

## Objective

Provide service businesses with safe, branded assistants that answer from approved knowledge, qualify enquiries, capture leads and direct users to configured booking or human-support channels.

## Initial supported capabilities

- Guided service selection
- Approved FAQ answers
- Deterministic price presentation
- Service-area guidance
- Lead capture
- Booking links
- WhatsApp escalation
- Unknown-question logging
- Responsive web interface

## Out of scope for the first release

- Autonomous technical diagnosis
- Unreviewed legal or regulatory advice
- Direct repair instructions
- Voice calling
- Unrestricted web search
- Automatic modification of approved knowledge

## Reference implementation

GetGasCert Assistant, identifier `RI-001`.
